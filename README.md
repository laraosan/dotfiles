# Larao Dotfiles

This repository is for managing configuration files (dotfiles) for my personal development environment.

## Install

First, git clone to your local environment.
```bash
git clone https://github.com/laraosan/dotfiles.git ~/.config/dotfiles
```

You can set up the dotfiles by running the following commands.
```bash
bash ~/.config/dotfiles/install.sh
```
Alternatively, you can create the symlink manually.
```
ln -s ~/.config/dotfiles/vimrc ~/.vimrc
```
